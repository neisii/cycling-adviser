# 사이클링 날씨 어드바이저

A Korean cycling weather advisor for 8 major Korean cities. Get a real-time suitability score, outfit recommendations, and a 3-day forecast before heading out on your bike.

## Features

- **0–100 suitability score** based on temperature, wind, precipitation, humidity, and UV index
- **5-level verdict** — 최상 / 좋음 / 보통 / 나쁨 / 위험
- **Outfit recommendations** with insulation, waterproof, and material attributes
- **3-day forecast** with per-day score and conditions
- **8 cities** — 서울, 부산, 인천, 대구, 광주, 대전, 울산, 제주
- Data aggregated from 3 weather providers for higher reliability

## Getting Started

### Prerequisites

- Node.js 24+
- pnpm 9+

### Install dependencies

```bash
pnpm install
```

### Environment variables

Create a `.env` file or set these in your shell:

| Variable | Description |
|----------|-------------|
| `WEATHER_PROXY_API_KEY` | API key for the weather proxy server |
| `SESSION_SECRET` | Secret for session signing |

### Run in development

Start the API server and the frontend dev server in separate terminals:

```bash
# Terminal 1 — API server (port 8080)
pnpm --filter @workspace/api-server run dev

# Terminal 2 — Frontend (port auto-assigned)
BASE_PATH=/ PORT=5173 pnpm --filter @workspace/cycling-advisor run dev
```

Then open `http://localhost:5173` in your browser.

## Deployment on Render.com

A `render.yaml` Blueprint is included at the repo root. The Express server builds the React SPA and serves it as static files — only one Render service is needed.

### One-time setup

1. Push this repo to GitHub.
2. Go to [render.com](https://render.com) → **New → Blueprint** and connect your repo.
3. Render detects `render.yaml` automatically and pre-fills the build/start commands.
4. Set the following environment variable manually in the Render dashboard:
   - `WEATHER_PROXY_API_KEY` — your proxy API key
5. Click **Deploy**.

`SESSION_SECRET` is auto-generated by Render. Everything else is configured in `render.yaml`.

### Build & start commands (reference)

```bash
# Build
npm install -g pnpm && \
pnpm install --frozen-lockfile && \
BASE_PATH=/ PORT=3000 pnpm --filter @workspace/cycling-advisor run build && \
pnpm --filter @workspace/api-server run build

# Start
node --enable-source-maps ./artifacts/api-server/dist/index.mjs
```

## Scoring Logic

| Factor | Ideal range | Max deduction |
|--------|-------------|---------------|
| Temperature | 10–24°C | –40 pts |
| Wind speed | < 6 m/s | –40 pts |
| Precipitation | None | –50 pts |
| Humidity | 30–80% | –15 pts |
| UV index | < 8 | –10 pts |

Thunderstorms, blizzards, and other extreme conditions override the score to 0 (위험).
